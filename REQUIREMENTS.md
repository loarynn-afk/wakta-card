# 🎴 왁타버스 멤버 카드 사이트

## 📋 프로젝트 개요
- **프로젝트명**: 왁타버스 멤버 카드 사이트 (가칭)
- **레퍼런스**: https://hearthstone.blizzard.com/ko-kr/cards
- **목적**: 왁타버스 멤버들을 카드 형태로 소개하는 팬메이드 사이트
- **협업**: 디자이너님 (디자인 에셋 담당) + btc (기능 구현 담당)

---

## 🎯 요구사항 정리

### 1. 상단 영역
- [ ] 메뉴탭 (선택사항 - 안되면 링크로 대체 가능)
- [ ] 배경 이미지
- [ ] 가운데 로고

### 2. 카테고리 필터 (핵심 기능)
- [ ] **고정멤버** / **중간계** / **고멤고멤** / **이세돌** / **왁타버스** / **ALL**
- [ ] 클릭하면 해당 카테고리만 표시
- [ ] (선택) 검색 기능

### 3. 카드 그리드
- [ ] 멤버 카드 이미지들을 그리드로 배치
- [ ] 반응형 (화면 크기에 따라 열 수 조정)

### 4. 카드 상세 팝업 (클릭 시)
- [ ] 큰 카드 이미지
- [ ] 멤버 소개글 (1~2줄)
- [ ] 유튜브 링크 버튼
- [ ] 숲(SOOP) 링크 버튼
- [ ] 닫기 버튼 (X)
- [ ] 배경 어둡게 처리 (오버레이)

## 🖼️ 레퍼런스 분석 (하스스톤 카드 라이브러리)

### 전체 레이아웃
```
┌─────────────────────────────────────────────────────────┐
│  [로고]  메뉴1  메뉴2  메뉴3  ...        [검색] [계정]   │ ← 상단 메뉴바 (어두운 배경)
├─────────────────────────────────────────────────────────┤
│                                                         │
│                    🖼️ 배경 이미지                        │
│                    + 로고 + 버튼                         │ ← 메인 배너
│                                                         │
├─────────────────────────────────────────────────────────┤
│  [필터1] [필터2]  [0][1][2]...[10+]  [검색창]  [필터]   │ ← 필터 영역 (적갈색 배경)
├─────────────────────────────────────────────────────────┤
│  1406개 카드 찾음                      정렬:  보기:     │ ← 정보 바
├─────────────────────────────────────────────────────────┤
│              ══════ 섹션 제목 ══════                    │
│                                                         │
│   🃏  🃏  🃏  🃏  🃏  🃏                                │ ← 카드 그리드
│   🃏  🃏  🃏  🃏  🃏  🃏                                │   (6열, 베이지 배경)
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 카드 상세 팝업
```
┌─────────────────────────────────────────────────────────┐
│                                                    [X]  │
│                                                         │
│        ┌────────┐    카드 이름                          │
│        │        │    플레이버 텍스트                     │
│        │  큰    │                                       │
│        │  카드  │    • 속성1: 값                        │
│        │  이미지│    • 속성2: 값                        │
│        │        │    • 속성3: 값                        │
│        └────────┘                                       │
│                                                         │
│   [◀]                                           [▶]    │ ← 좌우 네비게이션
│                                                         │
└─────────────────────────────────────────────────────────┘
(배경: 어둡게 오버레이)
```

## 🎨 디자이너님께 받을 에셋

### 필수
| 에셋 | 설명 | 예상 형식 |
|------|------|----------|
| 카드 이미지들 | 각 멤버별 카드 | PNG (투명 배경 권장) |
| 배경 이미지 | 메인 배너 배경 | JPG/PNG |
| 로고 | 사이트 로고 | PNG (투명 배경) |

### 선택
| 에셋 | 설명 | 예상 형식 |
|------|------|----------|
| 카테고리 아이콘 | 필터 버튼용 | PNG/SVG |
| 카드 뒷면 | 로딩 중 표시용 | PNG |
