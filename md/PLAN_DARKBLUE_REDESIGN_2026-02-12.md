## 수정 파일 목록 (7개)

### 1. globals.css - 테마 색상 변수 전체 변경
- 기존 갈색/베이지 변수 → 다크 블루 계열로 교체
- 입체감을 위한 글로우/섀도우 키프레임 애니메이션 추가

### 2. Header.tsx - 네이비 톤 + 하단 글로우 라인
- 배경: 네이비 그라데이션
- 하단 보더: 블루 글로우 효과

### 3. members/page.tsx - 배너 + 메인 콘텐츠 영역
- 배너: 다크 블루 + radial glow 효과
- 필터 섹션: 네이비 배경
- 메인 콘텐츠: 베이지 → 다크 네이비

### 4. CategoryTabs.tsx - 네이비 톤 탭
- 갈색 → 네이비 그라데이션
- 골드 악센트 유지

### 5. SearchBar.tsx - 네이비 톤 입력창
- 갈색 → 네이비 배경 + 블루 보더

### 6. ResultBar.tsx - 다크 배경용 텍스트 색상
- 다크 텍스트 → 라이트 텍스트로 변경
- 셀렉트박스 다크 테마

### 7. Footer.tsx - 네이비 톤 통일

## 입체감 강화 포인트
- 헤더 하단에 블루 글로우 라인
- 배너에 radial-gradient로 중앙 빛 효과
- 카드 호버 시 블루 글로우 강화
- 섹션 간 미묘한 그라데이션 차이로 깊이감
