### 세션 7: 탭/필터 섹션 분리
- **HTML 구조 변경**
  - 탭을 필터 섹션에서 배너 안으로 이동
  - 필터 섹션은 검색바만 포함

- **탭 스타일 변경 (CSS)**
  - `position: absolute` + `bottom: 0` + `left: 20px`로 배너 좌측 하단에 위치
  - 탭 아랫면이 필터 섹션 윗면과 맞닿음
  - 배너 `padding-bottom: 45px`로 탭 공간 확보

- **선택된 탭 스타일 변경**
  - 테두리만 노란색(gold)
  - 내부 색상은 다른 탭과 동일 (어두운 갈색)
  - 텍스트 색상만 노란색으로 변경

- **탭 호버 효과 변경**
  - 기존: `padding-top` 변경으로 모든 탭이 영향받음
  - 변경: `transform: translateY(-15px)`로 해당 탭만 위로 이동

- **탭 카드 스타일로 변경 (하스스톤 레퍼런스)**
  - 일반 탭: 87x50px, 선택된 탭: 87x60px
  - 탭이 필터 섹션 배경 뒤에 위치 (`z-index: 1`)
  - 필터 섹션이 탭 위로 덮음 (`z-index: 5`)
  - 글자를 탭 상단에 위치 (`align-items: flex-start`, `padding-top`)
  - 호버 시 15px 위로 떠오르는 효과
  - 반응형 스타일 업데이트 (768px, 480px)

- **필터 섹션 복구**
  - 갈색 배경 복구

- **정렬 드롭다운 디자인 변경**
  - 배경색: 흰색 → 양피지 색상 (`--color-parchment-dark`)
  - 테두리: 배경과 어울리는 갈색 계열
  - 호버/포커스 효과 추가
